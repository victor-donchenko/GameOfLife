Conway's Game of Life

Click -- toggle cell
Drag -- move screen
Space -- pause or unpause
Up/Down -- zoom in/zoom out
a/d -- speed up/slow down (changes milliseconds per update cap)
This is a Conway's Game of Life calculator. It uses a separate thread to do the grid updates to keep the grid update cap timings accurate, with synchronization to keep grid updates correct. It uses an optimized grid update algorithm to allow for quick updates when there are few on cells even with large grid sizes. The grid loops around at the edges.
